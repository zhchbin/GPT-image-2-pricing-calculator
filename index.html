GPT Image 2 图片生成成本估算工具

这个在线计算器用于估算 OpenAI gpt-image-2 的图片生成价格。你可以输入中文或英文 prompt，选择 low、medium、high 质量档位，设置图片宽高和生成张数，然后查看文本输入 tokens、单张输出 tokens、美元成本以及人民币换算结果。

当前 Standard 费率按文本输入 $5 / 1M tokens、图像输出 $30 / 1M tokens 计算。图片 output tokens 会受尺寸、长宽比例和质量档位影响，因此同样的 prompt 在 1024 x 1024、 1536 x 1024、2048 x 2048 等尺寸下会产生不同成本。

本工具适合在调用 OpenAI Images API 前做预算评估、批量生成报价、内容生产成本预估和 AI 图片服务定价参考。最终账单仍以 OpenAI API 返回的 usage 和官方账单为准。
